# RehabFlow

RehabFlow is a digital physiotherapy and rehabilitation support system designed for both patients and physiotherapists.

## Features (MVP)
- Patient-based exercise selection
- Region-based rehabilitation guidance (shoulder, knee, neck etc.)
- Disease-specific exercise programs
- Physiotherapist control panel (program creation)

## Purpose
To simplify home exercise prescription and improve rehabilitation adherence.

## Tech Stack (initial)
- HTML
- CSS
- JavaScript

## Future Plans
- User authentication
- Video-supported exercises
- AI-based exercise recommendations
- Mobile application
